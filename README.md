# APCSA-Sudoku-Project
Name: Andrew Gao
My Sudoku Generator Project for my APCSA class.

My Java program codes a completed Sudoku puzzle. It generates a 9×9 grid in which each row, column, and 3×3 subgrid contains the digits 1 through 9 without repetition.

Copy the code and paste it into an IDE or save the file and run it from a terminal on your PC

The Program loops through every element of an 2D Array from Left to Right, Top to Bottom. During every iteration it searches for available values for the element and then randomly sets one of the avaible values to the element of the 2D Array.
